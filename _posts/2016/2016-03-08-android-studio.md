## 官方IDE

android studio 是和eclipse类似的编辑器。  

官方自吹很厉害。  

* Intelligent code editor  
* Code templates  
* GitHub integration  
* Multi-screen app development  
* Virtual devices for all shapes and sizes  
* Android builds evolved, with Gradle
