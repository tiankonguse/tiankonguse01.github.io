curl 命令常用来拉取http数据。  

一般我们需要使用`-H`参数增加包头header。  


比如

```
curl -H "Host: data.video.qq.com"   'http://10.137.151.102/fcgi-bin/data?tid=786&appid=10001011&appkey=b26678a2d01bacb8&idlist=o0026evdykn&otype=json'
```
